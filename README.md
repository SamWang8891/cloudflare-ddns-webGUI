## Usage

1. Git clone to your choosen destination

   `git clone https://github.com/SamWang8891/cloudflare-ddns-webGUI.git`
2. run setup.sh

   `bash setup.sh` or `sudo bash setup.sh`
3. Visit the website on http://localhost:8888
4. Have fun!
