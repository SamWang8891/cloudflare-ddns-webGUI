## 使用方法

1. Git clone 到你的偏好位置

   `git clone https://github.com/SamWang8891/cloudflare-ddns-webGUI.git`
2. 執行 setup.sh

   `bash setup.sh` 或 `sudo bash setup.sh`
3. 於瀏覽器中連接 http://localhost:8888
4. 大功告成！
